## Install

Some basic Git commands are:

```
$ git clone https://github.com/bandara1997/Hardware_Management_System.git
$ cd project
$ npm install
```

## Start development

```
$ npm run dev
```

## Simple build for production

```
$ npm run build
```

## Run build for production

```
$ cd frontend
$ npm start
```
